<h1>Project requirements</h1>

## Client

- Node v20.11.0
- pnpm is used to install and run the project

```bash
    # Steps in order to run the client app

    npm install pnpm -g

    cd ./Client
    pnpm run dev
```

## Server

- .NET v8.0.101
- SQL Server

```bash
    # Steps in order to run the server app

    cd ./Server
    dotnet run
```
